SAARTOS

Features

  • Support SMP
  • Portable on Different Hardware Platforms
  • Interrupt Processing
    • Immediate mode
    • Deferred mode
  • Task Management
    • Static task creation
    • Dynamic task creation
  • Scheduling
    • Scheduler runs in O(1)
    • Fully-Partitioned Scheduling
    • Preemptive
    • Fixed-Priority - task priority can be changed when required by locking protocols
  • Memory Management
    • Partition-based
  • Tick & Time Management
    • Task Delay
    • Blocking Timeout
    • CPU Time Enforcement (in progress)
  • Synchronisation & Locking
    • spin-lock (ISR awared)
    • Task built-in Semaphore
    • Global Semaphore
    • Mutex lock with Immediate Priority Ceiling Protocol support
  • Inter-Task Communication
    • Messages and Queue
